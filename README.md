# Feedback Machine

A Chrome extension for leaving visual feedback — annotations, comments, and more — directly on any web page.

[**Install from Chrome Web Store**](https://chromewebstore.google.com/detail/EXTENSION_ID_HERE) | [**Website**](https://WEBSITE_URL_HERE)

## About This Repository

This is the **public issue tracker** for Feedback Machine. The extension's source code lives in a private repository.

Use this repo to:

- Report bugs
- Request features
- Browse and discuss existing issues

## Reporting Issues

We provide templates to help you submit useful reports:

- [**Bug Report**](https://github.com/onitheviking/feedback-machine/issues/new?template=bug_report.md) — Something broken? Let us know with steps to reproduce.
- [**Feature Request**](https://github.com/onitheviking/feedback-machine/issues/new?template=feature_request.md) — Have an idea? We'd love to hear it.

Please search [existing issues](https://github.com/onitheviking/feedback-machine/issues) before opening a new one to avoid duplicates.

## Security

If you discover a security vulnerability, **do not open a public issue**. Please follow the responsible disclosure process outlined in our [Security Policy](SECURITY.md).
